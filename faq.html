FAQ

What is Timesheet?

Timesheet is an easy to use, web based timesheet management system. Employees can enter/edit/submit their timesheets online anywhere, anytime. Managers in charge of the project will be able to view/approve/reject the timesheet submitted by employees. All users can access the system as long as they have a computer or phone and interview access.

What are the browsers Timesheet can work on?

Timesheet works best on modern Gecko and Blink/WebKit browsers. We recommend Google Chrome and Mozilla Firefox. However, all modern browsers should work with Timesheet.

Is maintenance and software updates included in the price?

Yes, there is no software version limiting on a subscription package. Any subscription package you purchase will always have free software updates and maintenance.

Is there a limit to the number of Jobs (Client Projects)/Users/Timesheet?

No, there are no limits to Timesheet’s number of jobs input, users registration and timesheets.

Can Timesheet generate reports?

Timesheet can generate the following reports:
  • Total hours by job (client project) and work type
  • Total hours per week per user per job (client project)
  • Leave usage
  • Total hours by user
  • Total hours by user for a week/fortnight
  • Total hours per job (client project)

Can Timesheet integrate with the payroll system I have?

Timesheet can integrate with iPayroll. It has the ability to export a CSV file and load into your existing payroll system.

Can I customize Timesheet to better suit my company’s needs?

Absolutely! From job management interfaces to task specification, Timesheet allows your company to manage your project information, your way.

How do I use Timesheet?

Timesheet is driven by a centralised web interface that can be accessed via your mobile device and desktop computer alike. The primary method of site navigation is via the sidebar, in which its options differ based on the type of account you have (i.e. Admin or Employee).
